```
#### Пример:
1. Мы вводим номер счета:
   ```
   print(def mask_account_card('Счет 64686473678894779589'))
   ```

   ```
   Счет **9589
   ```
   
или
2. Мы вводим номер и название карты:
   ```
   print(def mask_account_card('MasterCard 7158300734726758'))
